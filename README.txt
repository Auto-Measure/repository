Description du produit:
=====================================================================================
(English-version)


(French-version)
Auto Measure est une application intuitive qui permet aux utilisateurs de prendre leurs mensurations à domicile sans l'intervention d'un tailleur. Grâce à la caméra de l'appareil, l'application capture les dimensions nécessaires pour la confection de vêtements. Ensuite, les utilisateurs peuvent visualiser un modèle 3D d'eux-mêmes portant la tenue afin d'en évaluer l'ajustement et le style. Une fois satisfaits, ils peuvent envoyer leurs mensurations directement à leur tailleur. De plus, le tailleur pourra proposer différents modèles, qui seront affichés sur le modèle 3D de l'utilisateur, permettant ainsi de prévisualiser plusieurs options avant de faire un choix.
